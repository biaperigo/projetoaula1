1 - Abrir o vscode
2 - No terminal digitar: cd desktop
3 - No terminal dentro de desktop: mkdir projeto
4 - No terminal: cd projeto
5 - Dentro da pasta projeto: code .
6 - No terminal da pasta projeto: touch index.html
7 - No terminal: mkdir src
8 - No terminal: cd src
9 - No terminal: mkdir css
10 - No terminal: mkdir js
11 - No terminal: cd css
12 - No terminal: touch estilo.css
13 - No terminal: cd ..
14 - No terminal: cd js
15 - No terminal: touch script.js
16 - No terminal: cd ..
17 - No terminal: cd .. até chegar na raiz do projeto
18 - No terminal: git init
19 - Se estiver na Fiap: git config --global user.nome "Beatriz"
20 - Se estiver na Fiap: git config --global user.email biaperigo01@gmail.com
21 - No terminal: git status (Não so na FIAP)
22 - Se estiver vermelho: git add .
23 - Se estiver verde: git commit -m "mensagem do que você fez"
24 - Entrar no github em repositório, criar um novo repositório, colocar o nome e deixar publico, clicar no botão criar repositório (dica de ouro: nunca traduza a página do github)
25 - No bloco que estiver as 3 linhas:
  1° linha: conect seu gitbash com o seu github
  2° linha: muda o nome master para main
  3° linha: pega tudo que estiver no projeto e sobe para o github
26 - Caso você esteja em outro pc: no seu repositório no botão verde(code), copie o link 
27 - Abra o seu VSCODE no terminal cd desktop: git clone (cole o link), isso vai clonar seu projeto e pode continuar de onde parou
28 - Se modificar seu código faça todo o processo do
 git status,
 git add . 
 git commit -m "mensagem"
 e para subir as alterações só digitar : git push
